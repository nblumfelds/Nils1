About the page

  This page was created as a challenge. Previous to this I had no experience with HTML or CSS.

  I learned the basics using the w3school web page (link) and watching youtube videos. It was hard at the beginning but when I realized the way that an HTML file is structured it became a lot easier.

  I think that a modern engineer needs to understand a bit of computer coding since so much of our world has gone digital. HTML is not coding but it's a great stepping stone to get used to working with computer languages.

  The combination of a “regular” engineer (e.g. mechanical, electrical) with computer skills is an undeniable advantage in being competent and easily employable. That's why I want to develop skills in these areas.

  A great real life example could be Elon Musk. He developed computer skills from a young age and later went on to study physics.
